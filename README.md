# IPTV

HOW TO USE

1.OPEN SSIPTV SETTINGS IN TV 
2.GENERAR CONNECTION CODE "LAST CODE R932N"
3.COPY CODE FROM TV
4.LOGIN EM https://ss-iptv.com/
5.COPY CODE IN "ENTER CONNECTION CODE"
6.CLICK BUTTON ADD ITEM
7. IN DISPLAYED NAME IS THE CHANNEL NAME
